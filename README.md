# 欢迎使用K8SEASY

------

K8SEASY 是一个一键安装K8S高可用集群的软件。它可以帮助企业一键搭建完私有云系统，帮助用户在多家云服务商里灵活切换，不再被任何服务商绑架！ 对! 所有的操作只需要一键！


## 优点:

* 无需翻墙下载任何镜像
* 多节点高可用生产集群也只需要一键部署， 支持keepalived,haproxy 组合
* 无需任何配置
* 完美支持Centos7.0 和Ubuntu 16.04/18.04

更新日志
